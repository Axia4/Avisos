Conector Cordiax - Ntfy Notification Listener

Instructions:

1. Place 'ring.wav' and 'alarm.wav' in the same folder as the EXE.
2. The config file 'AjustesConectorCordiax.json' will be created in your user directory if not present.
3. Install dependencies for testing with Python (optional):
   pip install -r requirements.txt
4. To build EXE using PyInstaller:
   pyinstaller conector_cordiax.spec --onefile
